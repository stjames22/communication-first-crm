# BarkBoys Pilot Test Plan

## Goal
Validate that a team member can create an accurate quote quickly and consistently.

## Timebox
- Duration: 1 week
- Testers: 3-5 BarkBoys staff

## Test script (per tester)
1. Open estimator
2. Add 2-3 services from templates
3. Adjust quantity/zone/tax
4. Confirm preview total updates as expected
5. Save quote
6. Open PDF and verify values match

## Track these metrics
- Completion time (start to saved quote)
- Number of retries/errors
- Preview-to-saved mismatch incidents
- PDF generation failures
- Friction notes (free text)

## Pass criteria
- 90% of runs complete without blocker
- Median completion time <= 2 minutes
- 0 critical pricing mismatches

## Post-pilot decisions
- Keep as internal tool only, or
- Add roles/users + hosted deployment
